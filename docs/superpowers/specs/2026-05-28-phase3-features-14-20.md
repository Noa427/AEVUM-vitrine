## Feature 16 — WhatsApp Business

### Frontmatter customize.astro
Ajouter `GET /client/me` au `Promise.allSettled` existant (ou appel séparé) pour récupérer `whatsapp_connected: boolean`. La valeur est passée au template HTML via une variable SSR.

```ts
// Dans Promise.allSettled ou appel séparé
const meResult = await fetch(`${base}/client/me`, { headers, signal: AbortSignal.timeout(8000) });
const meData = meResult.ok ? await meResult.json().catch(() => ({})) : {};
const whatsappConnected: boolean = meData.whatsapp_connected ?? false;
```

### HTML — toggle WhatsApp par panel (customize.astro)
Ajouté dans chaque `.panel-right-actions` de TOUS les panels (onglets principaux + checkout abandon + témoignages + nouveaux panels F14/F15/F19), **sous** le toggle actif/inactif, **conditionnel** si `whatsapp_connected` :

```astro
{whatsappConnected && (
  <label class="toggle-switch toggle-wa" title="Envoyer aussi par WhatsApp">
    <input type="checkbox" class="toggle-wa-input"
           checked={parsedConfig.whatsapp_active ?? false} />
    <span class="toggle-track"></span>
    <span class="toggle-label">WhatsApp</span>
  </label>
)}
```

La valeur `whatsapp_active` est lue depuis le JSON parsé du config (nouveau champ dans `parseConfig`).

### Sauvegarde WhatsApp
Le JS `initEditor()` enrichit l'objet JSON du `hidden-value` au submit :
```js
const waInput = card.querySelector('.toggle-wa-input');
if (waInput) obj.whatsapp_active = waInput.checked;
```

Le toggle WhatsApp inline (changement sans submit) appelle `/api/config-update` via `fetch` identique au toggle actif/inactif existant.

Quand activé, révèle un bloc SMS sous l'éditeur principal :

## parseConfig — enrichissement

La fonction `parseConfig` existante retourne `{ subject, body, active, label? }`. Elle doit aussi retourner les nouveaux champs si présents dans le JSON :

```ts
function parseConfig(raw: string): {
  subject: string; body: string; label?: string; active: boolean;
  whatsapp_active?: boolean; sms_active?: boolean; sms_body?: string;
} {
  try {
    const p = JSON.parse(raw);
    if (p && typeof p.subject === 'string' && typeof p.body === 'string') {
      return {
        subject: p.subject, body: p.body,
        label: typeof p.label === 'string' && p.label ? p.label : undefined,
        active: p.active !== false,
        whatsapp_active: p.whatsapp_active === true,
        sms_active: p.sms_active === true,
        sms_body: typeof p.sms_body === 'string' ? p.sms_body : '',
      };
    }
  } catch {}
  return { subject: '', body: raw, active: true, whatsapp_active: false, sms_active: false, sms_body: '' };
}
```
